Pour utliser les scripts PHP, il faut récupérer le SDK Netatmo disponible ici : 
https://github.com/Netatmo/Netatmo-API-PHP
Il faudra placer les différents dossiers présents dans /src/ du SDK dans le dossier src de ce projet.

Il faudra également un serveur WEB qui hébergera tous ces fichiers. J'utilise pour ma part mon NAS qui est chez moi.
